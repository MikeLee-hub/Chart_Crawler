# MERN_Stack_Practice
Simple web project with MERN stack</br>
Crawling Billboard chart web sites

</br>

### frontend server 구동
In the project directory, you can run:

 `npm init`
 
 `npm install`
 
 `npm start`


### backend server 구동
In the project directory, you can run:

`cd backend`

`npm init`

`npm install`

`npm install -g nodemon`

`nodemon server`
